# 🐍 200+ Bài Tập Python Từ Cơ Bản Đến Nâng Cao

Chào mừng bạn đến với kho lưu trữ **200_bai_python**. Đây là dự án cá nhân tổng hợp hơn 200 bài tập lập trình Python, được thiết kế để rèn luyện tư duy thuật toán, cấu trúc dữ liệu và kỹ năng giải quyết vấn đề thực tế.

---

## 🚀 Mục tiêu của dự án
- 🧠 **Củng cố nền tảng:** Từ các kiểu dữ liệu cơ bản, vòng lặp đến xử lý hàm.
- 📉 **Thuật toán & Giải thuật:** Tập trung vào các bài toán logic và tối ưu hóa code.
- 📂 **Thực hành:** Ứng dụng Python vào xử lý file, toán học và tự động hóa.

---

## 📂 Cấu trúc Repository
Các bài tập được đánh số thứ tự từ `1.py` trở đi, giúp bạn dễ dàng theo dõi lộ trình học tập:

- **Cơ bản:** Làm quen với biến, điều kiện, vòng lặp và kiểu dữ liệu List/Dict.
- **Trung cấp:** Xử lý chuỗi, List Comprehension, Function và Module.
- **Nâng cao:** Thuật toán phức tạp, xử lý ngoại lệ và các bài toán thực tế.
